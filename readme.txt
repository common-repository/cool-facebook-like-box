== Description ==

Cool Facebook Like Box is a free plugin that help you to change and customize colors and styles of Facebook Like Box.

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.

If you have a page in Facebook about your blog and want to show the Facebook Like Box with the recent updates and fans, just activate this widget on **Appearance | Widgets** youâ€™ll see options of attaching widgets to your sidebar.

Don`t use this plugin on localhost, it doesn`t work ! Why ?  
Because Facebook won`t have access to your local server, you must upload it directly to your online space.
